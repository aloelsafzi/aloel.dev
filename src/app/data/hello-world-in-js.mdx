### 🐱 Persiapan

1. ✅ Sudah install **vscode** atau text editor lainnya
2. ✅ Sudah install **nodejs**

### 🔥 Mari kita ngoding 💻

1. Buat folder helloworld
2. Buka aplikasi text editor dan open folder helloworld
3. Buat file baru di dalam folder helloworld, contoh index.js. Kemudian ketikan code berikut

```javascript
console.log("Hello World");
```

4. Buka terminal atau cmd dan ketikan berikut:

```bash
node index.js
```

Maka Akan muncul di terminal dengan output **Hello World**
