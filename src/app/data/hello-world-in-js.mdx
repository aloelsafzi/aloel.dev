### 🐱 Persiapan

✅ Sudah install **vscode** atau text editor lainnya\
✅ Sudah install **nodejs**

### 🔥 Mari kita ngoding 💻

Buat folder helloworld\
Buka aplikasi text editor dan open folder helloworld\
Buat file baru di dalam folder helloworld, contoh index.js. Kemudian ketikan code berikut

```js:~/hello.js
console.log("Hello World");
```

Buka terminal atau cmd dan ketikan berikut:

```bash:~/
node index.js
```

Maka Akan muncul di terminal dengan output **Hello World**\
