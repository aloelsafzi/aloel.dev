### 🖊️ Pengertian Git

`Git` adalah sistem pengontrol versi (version control system) yang digunakan dalam pengembangan perangkat lunak untuk melacak perubahan dalam kode sumber selama pengembangan. Git dirancang untuk mengkoordinasikan pekerjaan antara programmer, tetapi dapat digunakan untuk melacak perubahan dalam setiap set file. Ini mendukung pembuatan cabang, pelabelan, dan operasi non-linear, menyediakan fitur seperti kontrol akses dan manajemen tugas yang dirancang untuk proses pengembangan perangkat lunak. Git dikembangkan oleh Linus Torvalds, pencipta sistem operasi Linux.

### 🔥 Setup Config Git

Ada 2 parameter yang pertama harus kita set pada config dan git kita di local yaitu `user.name` dan `user.email`, berikut perintah untuk melakukan config git nya:

```shell
git config --global user.name "jhon"
```

```shell
git config --global user.email "jhon@email.com"
```
